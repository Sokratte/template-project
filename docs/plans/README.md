# docs/plans/ — Plans

Plans are living rebuild guides, not disposable checklists. Steps are ticked as done; prose is kept truer than found. When fully complete, move to `docs/plans/archive/`.

**Naming:** `PLAN-NNN-topic-slug.md`
**Status lifecycle:** Draft → Plan ready → In Progress → Completed / Cancelled
