## Design decisions

- The template is its own showcase: real ADR, real spec, real research,
  real session logs, real changelog. Not just blank templates.
- LICENSE holder is "template-project contributors" — no personal names
  in a public template.
- README includes the full directory tree so people can see the structure
  before copying.
