# Historical Memory

*What the agent WAS. Never loaded automatically. The past tense of the
memory system — procedures that were retired and operational knowledge
that decayed out of active use. Append-only, never deleted, so the
history stays auditable.*

*The sweep writes decayed entries under `## stale operations`. Retired
procedures (a deliberate operator decision, never the sweep) go under
`## retired procedures`. These exact headings matter — the sweep script
uses them as insertion markers.*
